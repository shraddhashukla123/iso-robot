## Upload selected risks

```http
POST /risks/upload-selected
```

**Auth:** Bearer

Persists the analyst's chosen risks to the organisation's `risks` table.

### Request

```json
{
  "client_org_id": "uuid",
  "selected_risks": [
    {
      "issue_id": "uuid",
      "risk_title": "Quarterly access review gaps",
      "risk_rating": "Medium"
    }
  ]
}
```

### Response `200`

```json
{ "status": "success", "data": { "created": ["uuid"] } }
```
